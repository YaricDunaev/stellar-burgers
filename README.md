# Stellar Burgers 🍔

Веб-приложение для заказа бургеров с возможностью создания собственных рецептов в интерактивном конструкторе.

## 🚀 Демо

**[Открыть приложение на GitHub Pages](https://yaricdunaev.github.io/stellar-burgers/)**

## ✨ Возможности

- 🥖 **Конструктор бургеров** - перетаскивайте ингредиенты для создания собственного бургера
- 🔐 **Авторизация** - регистрация и вход в систему
- 📋 **История заказов** - просмотр всех ваших заказов
- 🎨 **Современный UI** - красивый и отзывчивый интерфейс
- 📱 **Адаптивный дизайн** - работает на всех устройствах

## 🛠 Технологии

- **React 18** - современный UI фреймворк
- **TypeScript** - типизированный JavaScript
- **Redux Toolkit** - управление состоянием
- **React Router** - навигация
- **Cypress** - E2E тестирование
- **Jest** - unit тестирование
- **Storybook** - документация компонентов

## 🚀 Быстрый старт

### Предварительные требования

- Node.js 18+
- npm или yarn

### Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/yaricdunaev/stellar-burgers.git
cd stellar-burgers

# Установка зависимостей
npm install

# Создание .env файла
cp .env.example .env

# Запуск в режиме разработки
npm start
```

Приложение будет доступно по адресу: http://localhost:4000

## 📝 Доступные команды

```bash
# Запуск в режиме разработки
npm start

# Сборка для продакшена
npm run build

# Запуск тестов
npm run test:jest

# Запуск E2E тестов
npm run test:e2e

# Запуск линтера
npm run lint

# Запуск Storybook
npm run storybook
```

## 🧪 Тестирование

Проект включает полный набор тестов:

- **Unit тесты** (Jest) - тестирование отдельных компонентов и функций
- **E2E тесты** (Cypress) - тестирование пользовательских сценариев
- **Storybook** - документация и тестирование компонентов в изоляции
